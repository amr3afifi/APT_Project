How To Run Crawler/Indexer    -> You will find instructions in the main class

1.Download XAMPP

2.Open XAMPP and start apache ,mysql, tomcat

3.make sure apache on ports 80,443 ..and mysql on port 3306 ..and tomcat on port 8080

4.Open page localhost/phpmyadmin and copy and paste the Structure script in the mysql folder in the project to create the database

5.Use the Fill scripts to fill the database with the data....Fill file for every table


 How to run web interface

1.Copy the folder "ROOT" found in ./Web-Interface and replace with ROOT folder in ./XamppPATH/tomcat/webapps

2.Open localhost:8080 on your browser
